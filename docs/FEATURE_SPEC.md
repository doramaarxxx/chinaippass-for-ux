# China IP Pass 기능명세서

## 1. 프로젝트 개요

### 1.1 서비스 설명
한국 특허사무소와 중견기업이 중국으로 특허/상표/디자인을 출원할 때 발생하는 복잡한 절차와 소통 문제를 해결하는 원스톱 IP 출원 플랫폼

### 1.2 해결하고자 하는 문제점
- 기존 한국-중국 특허사무소 간 이메일 소통 방식의 비효율성
- 복잡한 중국 IP 출원 절차로 인한 시간 소요 및 오류 발생
- 언어 장벽과 서류 작성의 어려움

### 1.3 타겟 사용자

| 사용자 유형 | 설명 |
|---|---|
| 한국 특허사무소 변리사 | 고객사를 대신하여 중국 IP 출원을 진행하는 전문가 |
| 중견기업 IP 담당자 | 직접 중국 IP 출원을 진행하는 기업 담당자 |

### 1.4 사용자의 목표
- 빠르고 정확한 중국 IP 출원 진행
- 복잡한 서류 작성 과정의 간소화
- 투명한 견적 확인 및 진행 상황 관리
