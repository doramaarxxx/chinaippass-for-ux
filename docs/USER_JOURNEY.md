**China IP Pass 사용자 여정**

---

**Journey 1. 신규 사용자 - 서비스 발견 및 가입**

1. 사용자가 검색 또는 광고를 통해 랜딩페이지에 진입한다
2. 히어로 섹션에서 "중국 IP 출원 원스톱 서비스" 핵심 가치를 확인한다
3. 스크롤하며 서비스 소개, 출원 유형(상표/특허/디자인), 장점 섹션을 훑어본다
4. CTA 버튼 "무료로 시작하기" 또는 헤더의 "회원가입"을 클릭한다
5. 회원가입 페이지에서 정보를 입력한다
   - 5-1. 이메일 입력 후 포커스 아웃 시 중복 확인이 자동으로 진행된다
   - 5-2. 비밀번호 입력 시 실시간으로 조건 충족 여부가 표시된다
   - 5-3. 사용자 유형(변리사/기업담당자)을 선택한다
   - 5-4. 필수 약관에 동의 체크한다
6. "회원가입" 버튼을 클릭한다
7. 이메일 인증 완료 안내를 확인한다 (Mock: 즉시 완료 처리)
8. 로그인 페이지로 자동 이동된다
9. 가입한 이메일/비밀번호로 로그인한다
10. 로그인 성공 시 랜딩페이지 또는 마이페이지로 이동한다

---

**Journey 2. 상표 출원 신청 (핵심 여정)**

**진입**
1. 로그인 상태의 사용자가 헤더 메뉴에서 "상표 출원"을 클릭한다
2. 상표 출원 Step 1 페이지로 이동한다
3. 상단에 4단계 Stepper가 표시된다 (현재: 1단계 활성)

**Step 1: 상표정보 입력**
4. 상표 유형을 선택한다 (문자상표 / 도형상표 / 결합상표)
   - 4-1. 도형 또는 결합 선택 시 이미지 업로드 영역이 나타난다
5. 상표명(한글)을 입력한다 (필수)
6. 상표명(영문)을 입력한다 (선택)
7. 상표명(중문) 입력란 옆 "자동번역" 버튼을 클릭한다
   - 7-1. 한글 상표명이 중문으로 자동 변환되어 채워진다
8. 중문 상표명 컨설팅이 필요한 경우 "중문 상표명 컨설팅 신청" 체크박스를 선택한다
   - 8-1. 체크 시 견적에 컨설팅 비용(150,000원)이 추가된다
9. 도형/결합상표인 경우 이미지를 업로드한다
   - 9-1. 드래그앤드롭 또는 클릭하여 파일 선택
   - 9-2. 업로드된 이미지 미리보기가 표시된다
   - 9-3. X 버튼으로 삭제 가능
10. 우선권 주장 여부를 선택한다 (예/아니오)
   - 10-1. "예" 선택 시 추가 필드가 나타난다
   - 10-2. 기초출원국 드롭다운에서 국가를 선택한다
   - 10-3. 기초출원일을 날짜 선택기로 입력한다
   - 10-4. 기초출원번호를 입력한다
11. "임시저장" 버튼 클릭 시 현재까지 입력 내용이 저장된다
12. "다음 단계" 버튼을 클릭한다
13. 유효성 검사 통과 시 Step 2로 이동한다
    - 13-1. 필수 항목 미입력 시 해당 필드에 에러 메시지가 표시된다

**Step 2: 상품 선택**
13. Stepper에서 2단계가 활성화된다
14. NICE 상품 분류 목록이 좌측에 표시된다 (01류 ~ 45류)
15. 검색창에 키워드를 입력하여 상품을 검색한다
    - 15-1. 검색어 입력 시 실시간으로 매칭되는 상품이 필터링된다
16. 원하는 류(Class)를 클릭한다
    - 16-1. 우측에 해당 류의 상품 목록이 체크박스와 함께 표시된다
17. 출원하려는 상품을 체크박스로 선택한다
    - 17-1. 선택된 상품이 상단 "선택된 상품" 영역에 태그로 추가된다
    - 17-2. 태그의 X 버튼으로 선택 해제 가능
18. 다른 류로 이동하여 추가 상품을 선택한다 (복수 류 선택 가능)
19. 목록에 없는 상품은 "추가 상품 입력" 태그 입력창에 직접 입력한다
    - 19-1. 엔터 또는 콤마로 태그 추가
20. 선택된 상품 수와 예상 비용이 하단에 실시간 표시된다
21. "임시저장" 버튼 클릭 시 선택 내용이 저장된다
22. "다음 단계" 버튼을 클릭한다
    - 22-1. 상품이 하나도 선택되지 않은 경우 에러 메시지가 표시된다

**Step 3: 견적 확인**
23. Stepper에서 3단계가 활성화된다
24. 상표 정보 요약이 표시된다 (상표명, 유형, 우선권 여부)
25. 선택된 상품 류(Class) 목록이 표시된다
26. 비용 상세 내역 테이블이 표시된다
    - 26-1. 기본 출원료 (1류)
    - 26-2. 추가 류 비용 (선택한 류 수에 따라)
    - 26-3. 우선권 주장 비용 (해당 시)
    - 26-4. 서비스 수수료
    - 26-5. 부가세
    - 26-6. 총 비용
27. 원화/달러 금액이 함께 표시된다
28. "견적서 다운로드 (원화)" 버튼 클릭 시 PDF가 다운로드된다
29. "견적서 다운로드 (달러)" 버튼 클릭 시 달러 버전 PDF가 다운로드된다
30. "이전 단계" 버튼으로 Step 2로 돌아가 상품을 수정할 수 있다
31. "다음 단계" 버튼을 클릭한다

**Step 4: 신청인 정보 & 제출**
32. Stepper에서 4단계가 활성화된다
33. 신청인 유형을 선택한다 (개인/법인)
    - 33-1. 법인 선택 시 사업자등록번호, 대표자명 필드가 추가로 표시된다
34. 신청인명(한글)을 입력한다
35. 신청인명(영문)을 입력한다
36. "자동번역" 버튼으로 신청인명(중문)을 자동 생성한다
37. 법인인 경우 사업자등록번호를 입력한다
38. 법인인 경우 대표자명을 입력한다
39. 연락처를 입력한다 (자동 하이픈 포맷팅)
40. 이메일을 입력한다
41. "주소 검색" 버튼을 클릭한다
    - 41-1. 도로명주소 검색 팝업이 열린다
    - 41-2. 주소를 검색하여 선택한다
    - 41-3. 선택한 주소가 주소(한글) 필드에 자동 입력된다
    - 41-4. 영문주소가 자동으로 변환되어 채워진다
42. 상세주소를 입력한다
43. 대리인 지정 여부 체크박스를 선택한다 (선택사항)
    - 43-1. 체크 시 대리인 정보 입력 필드가 나타난다
    - 43-2. 대리인명, 연락처, 이메일을 입력한다
44. 첨부서류를 업로드한다
    - 44-1. 법인인 경우 사업자등록증 업로드 (필수)
    - 44-2. 위임장 업로드 (선택)
    - 44-3. 우선권 주장 시 우선권증명서류 업로드 (필수)
45. 전자서명 동의 체크박스에 체크한다
46. "신청서 제출" 버튼을 클릭한다
47. 최종 검토 모달이 열린다
    - 47-1. 입력한 모든 정보가 요약되어 표시된다
    - 47-2. 내용을 확인한다
48. 모달에서 "제출 확인" 버튼을 클릭한다
49. 제출 완료 페이지로 이동한다
    - 49-1. 신청번호가 표시된다
    - 49-2. 안내 메시지가 표시된다 (담당자 검토 후 연락 예정)
50. "마이페이지로 이동" 또는 "새 출원 신청" 버튼을 클릭한다

**Journey 3. 특허 출원 신청**

**진입**
1. 로그인 상태의 사용자가 헤더 메뉴에서 "특허 출원"을 클릭한다
2. 특허 출원 Step 1 페이지로 이동한다
3. 상단에 2단계 Stepper가 표시된다

**Step 1: 특허서지사항 & 견적**
4. 특허 유형을 선택한다 (발명특허 / 실용신안)
5. 발명의 명칭(한글)을 입력한다
6. 발명의 명칭(영문)을 입력한다
7. "자동번역"으로 중문 명칭을 생성한다
8. 기술분야를 드롭다운에서 선택한다 (IPC 대분류)
9. 청구항 수를 입력한다
10. 도면 수를 입력한다 (선택)
11. 명세서 페이지 수를 입력한다
12. 발명자 정보를 입력한다
    - 12-1. 이름, 국적, 주소를 입력한다
    - 12-2. "발명자 추가" 버튼으로 복수 발명자를 추가한다
    - 12-3. 각 발명자 옆 삭제 버튼으로 제거 가능
13. 우선권 주장 여부를 선택한다
14. PCT 출원 여부를 선택한다
15. 입력 내용에 따라 견적이 실시간 계산되어 우측에 표시된다
16. "견적서 다운로드" 버튼으로 PDF를 받을 수 있다
17. "다음 단계" 버튼을 클릭한다

**Step 2: 신청정보 & 제출**
18. 상표 출원 Step 4와 동일하게 신청인 정보를 입력한다
19. 첨부서류를 업로드한다
    - 19-1. 명세서 (필수)
    - 19-2. 청구범위 (필수)
    - 19-3. 도면 (선택)
    - 19-4. 요약서 (필수)
    - 19-5. 우선권증명서류 (해당 시)
20. 전자서명 동의 후 제출한다
21. 제출 완료 페이지에서 신청번호를 확인한다

**Journey 4. 디자인 출원 신청**

**진입**
1. 로그인 상태의 사용자가 헤더 메뉴에서 "디자인 출원"을 클릭한다
2. 디자인 출원 Step 1 페이지로 이동한다

**Step 1: 디자인 정보 & 견적**
3. 디자인 명칭(한글/영문/중문)을 입력한다
4. 로카르노 분류를 선택한다
   - 4-1. 대분류 선택 후 소분류를 선택하는 2뎁스 구조
5. 디자인 이미지를 업로드한다 (최소 1장, 최대 10장)
   - 5-1. 드래그앤드롭으로 다중 업로드 가능
   - 5-2. 이미지 순서를 드래그로 변경 가능
   - 5-3. 각 이미지에 삭제 버튼 제공
6. 디자인 설명을 입력한다 (선택)
7. 창작자 정보를 입력한다 (복수 입력 가능)
8. 우선권 주장 여부를 선택한다
9. 견적이 실시간 계산되어 표시된다
10. "다음 단계" 버튼을 클릭한다

**Step 2: 신청인 정보 & 제출**
11. 상표 출원 Step 4와 동일하게 진행한다
12. 제출 완료 후 신청번호를 확인한다

**Journey 5. 임시저장 후 이어서 작성**

1. 사용자가 출원 신청 중 브라우저를 닫거나 이탈한다
2. 이후 다시 로그인한다
3. 마이페이지 > 신청 내역으로 이동한다
4. "임시저장" 상태의 신청 건을 확인한다
5. 해당 건의 "이어서 작성" 버튼을 클릭한다
6. 마지막으로 작성 중이던 단계로 이동한다
7. 기존에 입력한 데이터가 모두 복원되어 있다
8. 이어서 작성을 진행한다

---

**Journey 6. 신청 내역 확인 및 관리**

1. 로그인 후 헤더의 사용자 메뉴에서 "마이페이지"를 클릭한다
2. 마이페이지 > 신청 내역 페이지로 이동한다
3. 탭으로 출원 유형을 필터링한다 (전체/상표/특허/디자인)
4. 상태 필터로 원하는 상태의 건만 조회한다 (임시저장/접수완료/진행중/완료)
5. 검색창에 신청번호 또는 명칭을 입력하여 검색한다
6. 목록에서 원하는 신청 건을 찾는다
7. "상세보기" 버튼을 클릭한다
8. 상세 정보 모달 또는 페이지에서 전체 내용을 확인한다
   - 8-1. 상표/특허/디자인 정보
   - 8-2. 선택한 상품/분류
   - 8-3. 신청인 정보
   - 8-4. 첨부 파일 목록
   - 8-5. 현재 진행 상태
9. 임시저장 상태의 건은 "삭제" 버튼으로 삭제할 수 있다
   - 9-1. 삭제 확인 모달이 표시된다
   - 9-2. 확인 시 해당 건이 삭제된다

---

**Journey 7. 사용자 정보 관리**

1. 마이페이지 > 내 정보 관리로 이동한다
2. 현재 등록된 정보가 표시된다
3. "수정" 버튼을 클릭하여 편집 모드로 전환한다
4. 이름, 연락처, 소속을 수정한다
5. "저장" 버튼을 클릭한다
6. 수정 완료 토스트 메시지가 표시된다
7. 비밀번호를 변경하려면 "비밀번호 변경" 버튼을 클릭한다
   - 7-1. 현재 비밀번호를 입력한다
   - 7-2. 새 비밀번호를 입력한다
   - 7-3. 새 비밀번호 확인을 입력한다
   - 7-4. "변경" 버튼을 클릭한다
8. 회원탈퇴를 원하면 "회원탈퇴" 버튼을 클릭한다
   - 8-1. 탈퇴 확인 모달이 표시된다
   - 8-2. 비밀번호를 입력하고 확인한다
   - 8-3. 탈퇴 처리 후 로그아웃되어 랜딩페이지로 이동한다

---

**Journey 8. 어드민 - 신청서 처리**

1. 관리자 계정으로 로그인한다
2. 어드민 대시보드로 이동한다
3. 대시보드에서 오늘 접수된 신청 건수, 처리 대기 건수를 확인한다
4. 사이드바에서 "신청서 관리"를 클릭한다
5. 상태별 탭에서 "접수" 탭을 선택하여 신규 접수 건을 확인한다
6. 목록에서 처리할 신청서를 클릭하여 상세 페이지로 이동한다
7. 신청 정보를 검토한다
   - 7-1. 상표/특허/디자인 정보
   - 7-2. 신청인 정보
   - 7-3. 첨부파일 다운로드 후 확인
8. 상태 변경 드롭다운에서 "검토중"을 선택한다
9. 관리자 메모에 검토 내용을 작성한다
10. "저장" 버튼을 클릭한다
11. 검토 완료 후 "진행중"으로 상태를 변경한다
12. 출원 완료 후 "완료"로 상태를 변경한다
13. 필요 시 목록에서 CSV로 내보내기를 수행한다

**분기 시나리오**

**A. 로그인하지 않은 사용자가 출원 신청 시도 시**
1. 비로그인 상태에서 "상표 출원" 메뉴를 클릭한다
2. 로그인 페이지로 리다이렉트된다
3. 로그인 완료 후 원래 접근하려던 상표 출원 페이지로 이동한다

**B. 유효성 검사 실패 시**
1. 필수 필드를 비워두고 "다음 단계" 버튼을 클릭한다
2. 해당 필드에 빨간색 테두리와 에러 메시지가 표시된다
3. 페이지 최상단으로 스크롤되어 첫 번째 에러 필드가 포커스된다
4. 에러를 수정하고 다시 "다음 단계"를 클릭한다

**C. 이전 단계로 돌아가 수정 시**
1. Step 3(견적 확인)에서 "이전 단계" 버튼을 클릭한다
2. Step 2(상품 선택) 페이지로 이동한다
3. 기존 선택 내용이 유지되어 있다
4. 상품을 추가/삭제한다
5. "다음 단계"로 다시 이동하면 견적이 재계산되어 표시된다

**D. 세션 만료 시**
1. 오랜 시간 후 "다음 단계" 버튼을 클릭한다
2. 세션 만료로 인해 로그인 페이지로 이동한다
3. 다시 로그인한다
4. 임시저장된 데이터가 있다면 마이페이지에서 이어서 작성 가능
